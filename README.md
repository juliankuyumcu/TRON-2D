# TRON-2D

## Concept

A 2-D, 2-player snake-esque game based on the aesthetics of Tron.

### Warning

Old and unorganized.

## Usage

Launch ```tron.jar``` to start.

Click on headings to enter corresponding section.


Player 1: Arrow keys

Player 2: WASD keys
